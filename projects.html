Gravitational Lensing

Gravitational lensing is a prediction of Einstein's theory of general relativity, in which massive objects can act as lenses and bend light by inducing curvature in the spacetime around them. The image on the right shows Earendel, the most distant star ever observed, which was made visible by the gravitational lensing effect of a large galaxy cluster in the foreground. The star, along with the galaxy in which it resides, has been magnified into a long streak. The animation below is an attempt to recreate this image using a computational simulation. It shows how the lensed image changes as a simulated source galaxy is rotated. The animation was generated in python using the pyautolens library, with data from the Hubble Space Telescope. This kind of gravitational lensing is particularly exciting because it provides a way of directly observing the evolution of stars and galaxies in the early universe which would otherwise be too faint to see.

KerrGeoPy

KerrGeoPy is a python library for computing orbital trajectories around a spinning black hole. It implements the analytical solutions for plunging orbits from Dyson and van de Meent (arXiv:2302.03704), as well as solutions for non-plunging orbits from Fujita and Hikida (arXiv:0906.1420). The library also provides a set of methods for computing constants of motion and orbital frequencies, and can generate plots/animations like the ones shown here.
